Configuring DNS Records

This page explains how to configure DNS records to map multiple domains to a single website.

Before your mapped domains will work, you'll need to ensure that both your DNS records and server settings are configured properly.

DNS Records

Your DNS records should be configured the same as a normal website. Point the A records of any domains you want to map to your WordPress site's server.

We recommend using CloudFlare as your DNS host, but you can manage DNS records at your domain registrar or any other CDN or DNS provider you want.
