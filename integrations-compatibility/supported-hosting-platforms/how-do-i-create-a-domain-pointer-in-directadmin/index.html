How do I create a Domain Pointer in DirectAdmin?

  • Log in to the DirectAdmin control panel.

If you are hosting multiple domains, make sure you're modifying the settings for the proper domain by selecting it on the top of the DirectAdmin Dashboard:

Domain Selection if Using Multiple Domains

  • Find the Domain Pointers tool in the Dashboard:

Domain Pointers in DirectAdmin

  • Enter the Alias (mapped) domain into the Source field and ensure Create as an Alias is checked.

Creating an Alias Domain on DirectAdmin

  • The original website page should now appear for the alias domain.

Ensure your DNS Records are configured properly and that you have an SSL certificate configured for the Alias domain.
